偷来的上午

发表于 2018-01-21 
喧嚣  依旧

车潮  如往

在时光的间隙里

难得会有这么一个上午

不用披着沉沉月色

急步前行

带着和善的脸皮

亲近围墙高耸的宅院


这是一个难得的上午

一日之计于此伊始

抛却那些浮光掠影

我宁愿  捧本泛黄的书

立在阳台

贪婪嗅食着清新的空气

悄悄

心便随诗飞向了天际

飞离

这是非之地

负与痴

发表于 2018-01-21 
               《负》

孤枝几叶秋，疏月挂梢头，已是阴商长相望，莫断吾愁肠。

低首长吁叹，眉头蹙鬓头，已戍燕然三十年，何日相飞度?

               《痴》

漠云袭边廓，寒色侵衣钵，唯有死生相争渡，明日处何所?

凛风琢仪容，光华敲姿胸，且随归雁长安路，檐下重温故。

–源于我对古时闺怨以及边塞场景的想象，一唱一和。

赠曼殊

发表于 2018-01-21 
红馆绰约随影留，

蒲扇轻抚袈裟衣。

锦绣榻前禅机参，

佛祖容前红颜忆。

竹杖芒鞋醉樱雨，

苏堤隐梦常沾衣。

很不相逢百世前，

油灯檐下把酒彻。

读完白落梅写的《爱如禅 你如佛》，特作此篇。偶有败笔，请见谅。

你我停留

初次相见

颦笑永驻心田

我知

你是我注定的一半

固有的缄默

一视之光

洞穿你我场院

何时能牵你手腕

冲动的青春

执著红颜

难逃回眸瞬间

我惜

至此终须搁一段

写在雨季的邂逅

窣窣  重奏的是人海脚步

安排我们  如相约般

在雨季  重逢相遇

恍若隔世的余念

庇佑你的每一时 每一刻



或许我会可笑你的或许
